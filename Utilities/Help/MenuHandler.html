MenuHandler


Provide menu handling functionality to any object by storing menu objects which that object may want to create in order to remove them later. 


Add or remove a menu (a list of menu items) for an object. The object may be a class or instance. The object decides what the menu will be by responding to the message: menuItems. The method menuItems must return an array of CocoaMenuItems. Usually the object creates the menu items (thereby actually installing them on the Menu bar of SC) and then returns them. The Menu Handler stores these items as value in a Dictionary, using the object as key. This makes it possible for the object to remove the menu items by sending itself the message removeMenu. The method removeMenu looks the menu items up in the dictionary of MenuHandler, using the object as key, and lets MenuHandler remove those menu items.
